其他内容
id:唯一标识元素，在一个页面中不能重复,不同页面可以重复
name:可以用于定义一组标签，可以重复。 表单提交到后端处理的
style：定义标签的样式
class：定义标签的样式类
id，class都是css选择器和js的dom开发必备的基础
img标签和a标签一般都会设置title属性

标签类型---显示模式

行内元素：


行内元素不沾油独立的区域，仅仅靠自身的字体大小和图像尺寸来支撑结构。一般不可以设置宽度、高度、对齐等属性，常常用于控制页面中文本的样式
常见的行内元素： <strong> <b> <em> <i> <del> <s> <ins> <u> <a> <span>

块元素：


每个块元素都会占据一整行或者多整行，可以对齐设置宽度，高度，对齐等属性，常常用于网页布局和网页结构的搭建。
常见的块元素： <h1> // <p> // <div> // <ul> // <ol> // <li> //
