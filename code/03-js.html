pop和push组成栈的结构：先进后出

pop()从数组尾部弹出一个元素并返回弹出的元素，push从数组尾部压入一个元素，并返回数组的新长度

shift and unshift队列结构：先进先出

从数组的头部弹出一个元素，并返回次元素，索引为0，unshift从头部塞入一个元素 返回数组新长度

var t=[]
t.push(1)
t.push(2)
t.push(3)
console.log(t.pop())
console.log(t.shift())
console.log(t.unshift(4))

数组排序

sort() 排序是按照字符串方式进行比较，数字会改成字符串 所以数值排序要按照下面算

函数式编程

sort + 函数式编程

        要求：参数接收两个参数，要求返回值为：负数，0，正数。如果放回负数代表第一个参数小于第二个参数，0位相等，正数则大于。
        var comparefun= function(a,b){
            return a-b;
        };
        var m=[51,3,4,5,12,6];
        console.log(m);
        m.sort(m);
        console.log(m);
        利用函数式编程：
        m.sore(comparefun)
        更加简单的方法
        m.sort(function(a,b){return a-b;});

数组的连接方法

concat():连接原数组的元素和传递的参数形成一个新数组并返回，不影响原来的数组
不管后者之前是是什么，合并之后就是前者保持样子类型不变，后面的都是元素

join():把数组的元素连接成字符串，默认是逗号连接，括号内可以设置连接的方式""," ",","都是可以的

数组的切片

slice(): 对原数组没有任何影响截取数组的片段或者子数组；接受1or2个参数，开始索引，结束索引；如果2小于1，那么返回空数组；负数表示从后面切

splice()插入或者删除 会影响原来数组

1.接受1,2,3……个参数。2.1：删除数据的索引位置、2.删除数组元素的个数、3.开始要插入到原数组中的元素，插入的位置从第一个参数所在的索引开始
如果2号缺省，则表示从1号索引位置开始到最后都删光

冒泡排序法--降序

            var list=[2,4,2,5,7,8,5,6,3,1];
            var temp;
            for(var i=0; i < list.length; i++){
                for (var j=0; j <= i;j++){
                    if (list[j] < list[i]){
                        temp=list[i];
                        list[i]=list[j];
                        list[j]=temp;
                    }
                }
            }
            console.log(list);
